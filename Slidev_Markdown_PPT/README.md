# Getting Started

[Getting Started | Slidev](https://sli.dev/guide/#command-line-interface)

# Install slidev

Create a project folder:

```
npm init slidev@latest
```

# Run slidev

In the project folder:

```
npx slidev
```

# Try it Online

[sli.dev/new](https://sli.dev/new)
